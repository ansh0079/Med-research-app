PROBLEM: Searches showing "no results"?
CAUSE: Browser security blocks API requests when opening HTML files directly
SOLUTION: Use the local server (see below)

===============================================================================
  EASY SETUP (5 seconds)
===============================================================================

WINDOWS USERS:
  1. Double-click: START_SERVER.bat
  2. Browser opens automatically
  3. Start searching!

MAC/LINUX USERS:
  1. Double-click: start_server.sh (or run in terminal)
  2. Browser opens automatically
  3. Start searching!

ALTERNATIVE (any platform):
  1. Open terminal/command prompt in this folder
  2. Type: python run_server.py
  3. Press Enter
  4. Browser opens to http://localhost:8000

===============================================================================
  TROUBLESHOOTING
===============================================================================

Problem: "Python not found"
Solution: Install from python.org (check "Add to PATH" during install)

Problem: Server won't start
Solution: Open terminal in THIS folder first, then run the command

Problem: Still no results
Solution: 
  1. Open test-api.html to check connectivity
  2. Check browser console (F12) for error messages
  3. Make sure you're using the server, not opening HTML directly

Problem: AI Synopsis not working
Solution: Click Settings → Add your Anthropic API key
